# 目前測試出來的錯誤訊息

No suchkey 表示S3找不到檔案或是桶子名字輸入錯誤

PutItem error表示Dynamodb資料表名字或欄位輸入錯誤

# 可能的錯誤問題

Lambda輸入的S3桶子名稱錯誤(請確認S3桶子名稱，並參考步驟3進行問題排除)

Lambda輸入的Dynamodb錯誤(請確認Dyanmodb資料表，並參考步驟3進行問題排除)
